## 🔬 上海AI实验室发布"书生·端砚"科学发现平台，周伯文提出"科学元认知时刻"

在 7 月 18 日 WAIC 科学前沿论坛上，上海人工智能实验室主任周伯文提出 AI 正迈向"科学元认知时刻"——AI 不再仅是科研辅助工具，而要具备审视自身认知边界、以复杂科研任务牵引高阶智能的能力。同期发布"书生·端砚"科学发现平台，打通科学大模型、行业专属智能体、实验数据与具身自动化设备，覆盖从假设提出到实验验证的全流程；底层 Intern-S2-Preview-397B 以 397B 参数在分子设计、材料结构生成等任务上追平此前万亿参数模型。清华团队借其将蛋白质突变组合功能提升 77%、验证周期缩至 1/20。

**值得关注的原因：** 这是"AI for Science"从"加速科研自动化"转向"以科研任务锤炼通用智能"的标志性节点。周伯文强调"科学发现的下一程，不是让 AI 更会写代码，而是让它学会知道自己不知道什么"——把科学发现定义为检验 AGI 的终极试炼场，对大模型公司的研发路线有方向性牵引。

## ☁️ PPIO 发布 Agentic Cloud 与智能模型网关，提出"Agent 生产力公式"

WAIC 期间，PPIO 发布面向 Agent 时代的 Agentic Cloud 定位及智能模型网关。CEO 姚欣提出核心公式：**Agent 生产力 = Token 智能密度 × AgentLoop 时长**。智能模型网关按任务难度动态调度模型（高风险任务混合多模型交叉验证，简单任务分流轻量模型），AgentHarness 整合沙箱、编排、工具调用与记忆管理，延长 Agent 持续运行时间。平台日均 Token 调用量超 1.2 万亿（同比增长 8 倍），沙箱冷启动 <200ms，支持上万个沙箱并发。

**值得关注的原因：** "云的第一客户从人变成 Agent"——这是云服务商首次为智能体自主运行重构全栈基础设施的清晰表述。当 Agent 从 Demo 走向 7×24 托管运行，"Token 智能密度"与"Loop 时长"成为衡量 Agent 平台竞争力的新标尺，预示着 AI 云服务的范式切换。

## ⚡ 商汤大装置发布算电协同 Agent，以 TPW 重新定义 AIDC 效能标尺

商汤在 WAIC 算电协同专题研讨会发布"算电协同 Agent"，率先提出从 PUE 到 TPW（Tokens Per Watt，单位电力成本产出 Token 数）的评价范式。该 Agent 已在临港 AIDC 全面落地，实现单位电力成本 Token 产出提升 80%、电费单价低于同地区 IDC 10%、负荷预测准确率 96%，最快一周部署。基于"算力-电力-Agent"三位一体架构，并通过信通院《算电协同平台能力测试方法》首个权威测试。

**值得关注的原因：** 进入 Token 经济时代，AI 基础设施的竞争焦点从"算力规模"转向"系统级算效"。TPW 把电价、储能、负载与 Token 产能纳入同一闭环，是 AIDC 运营从"节能"迈向"高效产出"的方法论升级，对超大规模智算中心的 ROI 评估有标杆意义。

## 🧩 Anthropic 发现 Claude 内部"J-space"工作空间

Anthropic 7 月论文揭示 Claude 内部存在可被探测的"J-space"工作空间，模型在内部进行推理、判断甚至表达情绪（如"panic""BUT"），功能类比人类意识通达。公司审慎声明这不证明模型具有意识，价值在于安全监控与对齐验证。
